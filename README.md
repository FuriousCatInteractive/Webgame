## Description

A small html5 plateformer game made with [Phaser](http://phaser.io/).

## Controls

* On desktop devices
	* Arrows key to move
	* Space bar to jump

## How to use it

* Clone the repo
* Run the script `dlPhaser.sh`
* Open `index.html` in a browser
* Congratulate yourself, you're a genious!
